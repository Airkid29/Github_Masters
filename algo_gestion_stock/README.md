# Application de Gestion de Stock (Algorithme)

## Description
Ce projet présente la **logique algorithmique** d’une application de gestion de stock.  
Il a pour objectif de vérifier la maîtrise des **concepts fondamentaux de l’algorithmique** à travers un cas réel.

Aucune interface graphique ni langage de programmation spécifique n’est utilisé.  
Le travail est exprimé en **pseudo-code (algorithme normal)**.

---

## Objectifs pédagogiques
- Manipuler des variables et structures de données
- Utiliser des conditions et des boucles
- Gérer des entrées et sorties de stock
- Vérifier la cohérence des données
- Simuler des situations réelles (rupture, suppression, alerte)

---

## Fonctionnalités
- Ajouter un produit
- Approvisionner un produit
- Sortir un produit du stock
- Supprimer un produit
- Afficher l’état du stock
- Détecter les produits en seuil critique

---

## Structure logique
- Boucle principale avec menu
- Traitement par choix (`SELON`)
- Vérification systématique des erreurs
- Messages explicites pour chaque action

---

## Utilisation
Ce pseudo-code peut être :
- analysé pour un devoir
- traduit en Python, C, Java, etc.
- utilisé comme base pour une application réelle
